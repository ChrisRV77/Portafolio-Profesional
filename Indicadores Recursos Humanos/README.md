# Indicadores Recursos Humanos

El siguiente reporte busca replicar un sistema de base de datos usados durante un proyecto de indicadores de recursos humanos para una empresa que dirigía varias empresas y se necesitaba de un sistema automatizado para su presentación. EL trabajo presentado no representa el reporte final entregado, es una versión propia mía y con un enfoque diferente al usado. 

Todas las tablas presentadas fueron realizada por mí y otra parte sacada de una base de datos de información personal de libre acceso.

Para este reporte se tiene de base lo siguiente:
- Una **base de datos de acciones** (aqui se registra las acciones de ingreso y salidas de los empleados)
- Bases de dimensiones para cada relación en la base principal

### Portada
![Portada](https://github.com/ChrisRV77/Portafolio-Profesional/blob/main/Indicadores%20Recursos%20Humanos/Imagenes/Portada.png)
Para la portada dado que es un reporte recurrente que según la fecha en que se quiera presentar deberá resaltar cierto periodo de tiempo incluyo un segmentador para indicar hasta que año se querrá presentar la información del resto de las páginas.

### General
![General](https://github.com/ChrisRV77/Portafolio-Profesional/blob/main/Indicadores%20Recursos%20Humanos/Imagenes/General.png)

Para la primera diapositiva presentamos en la parte de arriba la segmentaciones de las empresas dentro del conglomerado para que así se pueda visualizar cada datos por empresa.

En el lienzo se encuentra los valores de ingreso y salida de empleados general y además su variación con el año pasado y en la parte de abajo una pequeña descripción de los que es la empresa con cierto hitos del año en cuestión que indican los resultados del incremento o la disminución de la cantidad de empleados.

### Rotación

![Rotación](https://github.com/ChrisRV77/Portafolio-Profesional/blob/main/Indicadores%20Recursos%20Humanos/Imagenes/Rotaci%C3%B3n.png)

En cambio, para el panel de rotación tenemos el valor de la rotación dado en el año, una segmentación dad por el tipo de salidas, además podemos comparar la cantidad de salidas con los años anteriores, también se cuenta con la rotación acumulada por meses y clasificada por año para que su crecimiento pueda ser comparado.
